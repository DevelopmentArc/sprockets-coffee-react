= sprockets-coffee-react

== How to use

Add this to your Gemfile:

  gem 'sprockets-coffee-react'

Place a <tt>.js.coffee.cjsx</tt> or a <tt>.js.coffee</tt> file with a <tt>@cjsx</tt> pragma in your assets directory.

Require within your <tt>application.js</tt> or somewhere else:

  //
  //= require my-component
  //

Done.
